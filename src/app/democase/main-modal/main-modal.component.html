模态框包含专门针对特定任务的文本和UI控件。 它们向用户通知关键信息，要求用户作出决定或涉及多个任务

API 选项

free-modal

单个模态框。

属性 类型 描述 header String 模态框标题。 visible String 是否显示模态框，默认为false theme String 设置预设样式，可选。可选值(primary/info/success/warning/danger) type String 设置模态框类型(alert,confirm,prompt)，可选。 spinner String 设置Loading类型，可选。 delay Number 设置Loading自动隐藏时间，可选。

Import

import {ModalModule} from 'freeng/freeng';
