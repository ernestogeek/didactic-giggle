上下文菜单(ContextMenu)在其目标上右击时，会显示一个重叠菜单。

API 选项

free-contextmenu

上下文菜单

属性 类型 描述 target String 需要上下文菜单的目标(一般使用模板变量) menu Array 一个菜单数组 global Boolean 设置document的上下文菜单，默认为false back Boolean 是否点击返回上一个路由页面，默认为false

Import

import {ContextmenuModule} from 'freeng/freeng';

Default

<free-contextmenu [target]="cm" [menu]="menu"></free-contextmenu>

Javascript

this.menu = [ {label: '子菜单', icon: 'user', item: [{label: '子菜单'}]}, {separator: true}, {label: '子菜单', icon: 'user', disabled: true}, {label: '子菜单', icon: 'user', item: [{label: '子菜单'}]} ];
