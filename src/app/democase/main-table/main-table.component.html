数据表可用于显示数据列表。

  • 分页，即时搜索和排序
  • 提供了丰富的支持,复选框

API 选项

free-table

必选，类似HTML的table，一个free-table对应一个table。

free-table-header

必选，类似HTML的thead，一个free-table-header对应一个thead。

free-table-body

必选，类似HTML的tbody，一个free-table-body对应一个tbody。

free-table-row

必选，类似HTML的tr，一个free-table-row对应一个tr。

free-table-cell

必选，类似HTML的td，一个free-table-cell对应一个td。

属性 类型 描述 colspan Number 规定单元格可横跨的列数，可选 rowspan Number 规定单元格可横跨的行数，可选

import {TableModule} from 'freeng/freeng';
