线性进度条填充0％至100％以显示任务的进度。 它还将动画显示有一个任务等待完成。

  • 丰富的颜色选择
  • 酷炫的动画效果

API 选项

free-progress

代表一个进度条

属性 类型 描述 value String 当前加载值 striped Boolean 是否增加斑马条纹，默认为false theme String 设置预设样式，可选。可选值(primary/info/success/warning/danger) move Boolean 是否增加动态效果，默认为false

Import

import {ProgressModule} from 'freeng/freeng';
