标签页可以轻松的切换应用的不同视图或功能方面，或者浏览分类数据集。

API 选项

free-tab-group

必选，表示一组Tab。

属性 类型 描述 direction String 设置Tab头的位置，默认在左侧.可选值(right, center)

free-tab

一个Tab。

属性 类型 描述 header String 标签标题，可选 disabled Boolean 是否禁用，可选，默认false

事件

事件名 类型 描述 onChange Function 当Tab切换时触发的回调函数。

Import

import {TabGroupModule} from 'freeng/freeng';

Direction

标签1 标签2 <free-tab-group direction="right"> <free-tab header="标题1"> 标签1 </free-tab> <free-tab header="标题2"> 标签2 </free-tab> </free-tab-group>

Disabled

标签1 标签2 标签3 <free-tab-group direction="right"> <free-tab header="标题1"> 标签1 </free-tab> <free-tab header="标题2" [disabled]="true"> 标签2 </free-tab> <free-tab header="标题3"> 标签3 </free-tab> </free-tab-group>

onChange

当前tab：{{activeState?.activeIndex}}

标签1 标签2 <free-tab-group (onChange)="onTabChange($event)"> <free-tab header="标题1"> 标签1 </free-tab> <free-tab header="标题2"> 标签2 </free-tab> </free-tab-group>
Javascript
onTabChange(event: any) { this.activeState = { activeIndex: event.activeIndex } }
