输入允许用户输入文本，通常以表单形式出现。用户可以输入文字，数字或混合格式类型的输入。

API 选项

free-inputtext

单个输入表单

属性 类型 描述 theme String 设置预设样式，可选。可选值(primary/info/success/warning/danger) placeholder String 设置提示信息，可选 pattern String 设置正则匹配信息，可选。
可选值：
  • tel:手机号
  • email:邮箱
  • card:身份证号码
  • chinese:中文
注：如果值不在内置验证中，会使用该值作为验证信息 message String 设置验证失败时的提示信息，可选 icon String 是否增加图标，可选(采用FontAwesome)

Import

import {InputtextModule} from 'freeng/freeng';
