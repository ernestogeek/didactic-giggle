数据表格可用于显示数据列表。

  • 分页，固定头和排序
  • 提供了丰富的支持,复选框

API 选项

free-datatable

必选，类似HTML的table，一个free-datatable对应一个table。

属性 类型 描述 value Array 数据数组，必选 border Boolean 是否显示边框，默认为false striped Boolean 是否显示斑马线，默认为false hover Boolean 是否添加悬停效果，默认为false selectionMode String 是否添加复选框或单选框，可选。可选值(single/multiple) pagination Boolean 是否显示分页器，可选，默认为false row Number 每页数据数，可选，默认为10 expandableRows Boolean 是否展示可折叠内容，默认为false。需结合ng-template，可参考右侧例子。 scrollable Boolean 是否固定头，默认为false。

free-datatable-column

必选，类似HTML的td，表示一列数据。

属性 类型 描述 field String 列的名称，必须，对应free-datatable里value数据里的键 header String 列的描述名称，必选 sort Boolean 当前列是否可排序，默认为fasel
