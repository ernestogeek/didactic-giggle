复选框允许用户从一组中选择多个选项。

  • 多种可选的主题样式

API 选项

free-checkbox

单个复选框

属性 类型 描述 label String 自定义文本，可选 name String 设置复选框的名称，可选 checked Boolean 是否默认选中，默认为false，可选 disabled Boolean 是否禁用，默认为false，可选 theme theme 设置预设样式，可选。可选值(primary/info/success/warning/danger)

Import

import {CheckboxModule} from 'freeng/freeng';

Default

<free-checkbox label="多选框" [checked]="true"></free-checkbox> <free-checkbox label="多选框"></free-checkbox>

Theme

<free-checkbox theme="primary" label="primary" name="theme" [checked]="true"></free-checkbox> <free-checkbox theme="info" label="info" name="theme"></free-checkbox> <free-checkbox theme="success" label="success" name="theme"></free-checkbox> <free-checkbox theme="warning" label="warning" name="theme"></free-checkbox> <free-checkbox theme="danger" label="danger" name="theme"></free-checkbox>

Disabled

<free-checkbox [disabled]="true" label="disabled checkbox"></free-checkbox> <free-checkbox [disabled]="true" label="disabled checkbox" [checked]="true"></free-checkbox>

Group

选择的城市：{{city}}

<free-checkbox value="广东" [(ngModel)]="selectedCity" label="广东"></free-checkbox> <free-checkbox value="北京" [(ngModel)]="selectedCity" label="北京"></free-checkbox> <free-checkbox value="上海" [(ngModel)]="selectedCity" label="上海"></free-checkbox>

Javascript

this.selectedCity = ['广东'];
