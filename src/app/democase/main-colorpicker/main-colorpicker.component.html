ColorPicker是用于选择颜色的输入组件。

API 选项

free-color-picker

颜色选择器

属性 类型 描述 colorInput Boolean 是否显示输入框，默认为false inline Boolean 是否内联显示，默认为false。

事件

事件 参数 描述 onChange {value: '#000', rgb: 'rgb(0,0,0)', hsl: 'hsl(0, 0%, 0%)'} 当选定颜色时的回调事件

Import

import {ColorpickerModule} from 'freeng/freeng';

Default

 <free-color-picker [(ngModel)]="defaultColor"></free-color-picker>

Input

 <free-color-picker (onChange)="onColorSelect($event)" [(ngModel)]="color" [colorInput]="true"></free-color-picker>

Javascript

this.color = 'rgb(110, 11, 248)'; onColorSelect(event: any) { console.log(event); }
