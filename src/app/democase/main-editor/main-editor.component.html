API 选项

free-editor

富文本编辑器

属性 类型 描述 toolbar boolean 是否显示工具栏，默认为true。 buttons Array 工具栏显示按钮，可选。 (bold/italic/underline/strikethrough/subscript/superscript/ heading/fontSize/foreColor/backColor/align/insertOrderedList/ insertUnorderedList/blockquote/table/createLink/insertImage/emotion/ fullscreen/undo/redo) maxHeight Number 设置最大高度 counter boolean 是否显示字数，可选，默认为false style object 设置编辑器样式，可选。
