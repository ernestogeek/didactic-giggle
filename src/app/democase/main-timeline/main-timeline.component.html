垂直展示的时间流信息。

API 选项

free-timeline

时间轴。

free-timeline-item

时间轴的每一个节点。

属性 类型 描述 header String 节点描述标题，可选。 dot String 节点图形，默认圆形，可选(square)

Import

import {TimelineModule} from 'freeng/freeng';
