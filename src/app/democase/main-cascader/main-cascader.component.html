级联选择器，可以从一组相关联的数据集合进行选择，例如省市区，事物分类等。

API 选项

free-cascader

级联选择器

属性 类型 描述 options Array 设置数据源，必选 pholder String 设置可描述输入字段预期值的提示信息，可选 separator String 设置分隔符，可选。默认是"/" deploy Boolean 输入框的长度是否随着选择值的长度而展开，可选。默认为false

回调事件

事件 描述 onChange 选择完成后的回调，可选。返回值格式： {{'{value: ["第一级值", "第二级值"...], label: "以字符串显示的选择值"}'}}

Import

import {CascaderModule} from 'freeng/freeng';

Default

{{userCity}}

 <free-cascader [options]="food" [(ngModel)]="selectFood" pholder="选择菜系"></free-cascader>
