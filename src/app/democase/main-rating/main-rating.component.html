星星评分

API 选项

free-rating

单个星星评分框。

属性 类型 描述 value Number 设置当前值。 theme String 设置预设样式，可选。可选值(primary/info/success/warning/danger) readonly Boolean 是否只读，可选。 onChange Function 监听change事件

Import

import {RatingModule} from 'freeng/freeng';

Default

<free-rating></free-rating>

Theme

<free-rating theme="primary" [value]="3"></free-rating> <free-rating theme="info" [value]="3"></free-rating> <free-rating theme="success" [value]="3"></free-rating> <free-rating theme="warning" [value]="3"></free-rating> <free-rating theme="danger" [value]="3"></free-rating>

Change Event

<free-rating (onChange)="ratingChange($event)"></free-rating>

Readonly

<free-rating [readonly]="true" [value]="3"></free-rating>

Hover

<free-rating [hover]="true" [value]="3"></free-rating>
