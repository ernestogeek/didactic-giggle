简单的页面消息通知组件。 可以设置让消息在页面的四个角落出现。可以手动关闭消息，也可以设置成自动关闭。

API 选项

free-notification

消息通知框。

属性 类型 描述 messages Array 消息数组。 delay Number 消息通知框延迟关闭时间。默认不关闭 direction String 消息通知框的位置。默认为“TopRight”。(可选值：topLeft, topRight, bottomLeft, bottomRight)

Import

import {NotificationModule} from 'freeng/freeng';

Default

<button fButton (click)="open()">消息通知</button> <free-notification [messages]="messages"></free-notification>

Javascript

open() { this.messages.push({ title: '通知', content: '你的帐号未经过邮箱验证！' }); }

Theme

<free-notification [messages]="errors" theme="error"></free-notification> <free-notification [messages]="infos" theme="info"></free-notification> <free-notification [messages]="successes" theme="success"></free-notification> <free-notification [messages]="warnigs" theme="warning"></free-notification>

Avatar

<free-notification [messages]="avatarMessages"></free-notification>

Javascript

openAvatarMessage() { this.avatarMessages.push({ title: '通知', content: '你的帐号未经过邮箱验证！' + new Date(), avatar: 'assets/images/t_me.png' }); }

Delay

<free-notification [messages]="delayMessages" [delay]="3000"></free-notification>

Direction

 <free-notification [messages]="toolMessages" [direction]="direction['value']" [delay]="1500"></free-notification>
