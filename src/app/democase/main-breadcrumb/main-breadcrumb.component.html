面包屑结构简单，支持自定义分隔符。

API 选项

free-breadcrumb

面包屑组。

属性 类型 描述 menus Array 面包屑数组 separator String 自定义分隔符 icon String 使用FontAwesome字体图标来自定义分隔符

Import

import {BreadcrumbModule} from 'freeng/freeng';

Default

<free-breadcrumb [menus]="menus"></free-breadcrumb>

Separator

<free-breadcrumb separator="|" [menus]="menus"></free-breadcrumb>

Icon

<free-breadcrumb icon="tag" [menus]="menus"></free-breadcrumb>
