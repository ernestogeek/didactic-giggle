图表组件是基于Chart.js， 一个基于开源的基于HTML5的图表库。

Chart.js

为了让图表控件生效，必须引入chart.js，有两种方式：

  • 通过外链
  • 通过CLI

外链chart.js

<script src="<Your_Path>/Chart.js"></script>

CLI

"scripts": [ "../node_modules/chart.js/dist/Chart.js" ]

API 选项

free-chart

图表控件

属性 类型 描述 width number 图表宽度，可选。 height number 图表高度，可选。 type string 图表类型，必选。 data any 图表数据结构，必选。 options any 图表配置数据，可选。

事件

事件 类型 描述 onDataSelect Function 当点击图表时触发的回调事件，可选。

Import

import {ChartModule} from 'freeng/freeng';

Bar

<free-chart type="bar" [data]="barData"></free-chart>

Line

<free-chart type="line" [data]="lineData"></free-chart>

Pie

<free-chart type="pie" [data]="pieData"></free-chart>

Doughnut

<free-chart type="doughnut" [data]="doughnutData"></free-chart>
