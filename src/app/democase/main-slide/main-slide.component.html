强大的触摸滑动幻灯片

API 选项

free-slides

一个幻灯片列表容器。

属性 类型 描述 styles Object 设置幻灯片样式，可选 pagination Boolean 是否显示分页器，可选 paginationClickable Boolean 分页器是否可点击，可选 arrow Boolean 是否显示左右切换箭头，可选 loop Boolean 是否可循环，可选 autoplay Boolean 是否自动播放，可选 autoplayDisableOnInteraction Boolean 用户操作之后，是否禁止autoplay。默认为true：停止。 如果设置为false，用户操作之后自动切换不会停止，每次都会重新启动autoplay。 操作包括触碰，拖动，点击pagination等。

free-slide

一个切换的幻灯片，可以包含文字、图片、html元素。

Import

import {SlideModule} from 'freeng/freeng';
