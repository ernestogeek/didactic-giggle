数字输入框控件，可直接输入数字，也可以点击“+”、“-”按钮变换当前数值。

API 选项

free-spinner

数字输入框

属性 类型 描述 value Number 设置当前值，默认为0 min Number 允许使用的最小值，默认无限制 min Number 允许使用的最大值，默认无限制 step Number 每次点击“+”、“-”按钮变化的步长，默认步长为1

Import

import {SpinnerModule} from 'freeng/freeng';

Default

<free-spinner></free-spinner>

Step

<free-spinner [step]="5"></free-spinner>
