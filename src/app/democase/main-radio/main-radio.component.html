单选按钮允许用户从一组中选择一个选项

API 选项

free-radio

一个Radio组件

属性 类型 描述 label String 自定义文本，可选 name String 设置单选框的名称，可选 checked Boolean 是否默认选中，默认为false，可选 disabled Boolean 是否禁用，默认为false，可选 theme theme 设置预设样式，可选。可选值(primary/info/success/warning/danger)

Import

import {RadioModule} from 'freeng/freeng';

Default

<free-radio label="男" name="sex" [checked]="true"></free-radio> <free-radio label="女" name="sex"></free-radio>

Theme

<free-radio theme="primary" label="primary" name="theme" [checked]="true"></free-radio> <free-radio theme="info" label="info" name="theme"></free-radio> <free-radio theme="success" label="success" name="theme"></free-radio> <free-radio theme="warning" label="warning" name="theme"></free-radio> <free-radio theme="danger" label="danger" name="theme"></free-radio>

Disabled

<free-radio label="disabled" [disabled]="true"></free-radio> <free-radio label="disabled" [disabled]="true" [checked]="true"></free-radio>
