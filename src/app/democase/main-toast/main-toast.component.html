Toast是一种消息提示框，可提供成功、警告和错误等反馈信息。 通常情况下，Toast会在短时间内显示，然后自动关闭。

API 选项

free-toast

消息提示框

属性 类型 描述 messages Array 消息数组 delay Number 消息延迟关闭时间，默认为1500(单位：ms) maxMessage Number 显示最大的消息数，默认为10 theme String 设置预设样式，可选。可选值(primary/info/success/warning/danger)

事件

事件 类型 描述 onClose Function 当消息关闭时触发的回调事件。
