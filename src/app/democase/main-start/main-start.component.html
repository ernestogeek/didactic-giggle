FreeNG是一款完全响应式的前端UI框架， 它采用了主流的左右两栏式布局，使用了Html5+CSS3+Angular2(+4)等流行的前端技术， 同时提供了诸多强大的可以重新组合的UI组件，门槛极低，拿来即用。

FreeNG还很年轻，首个版本发布于2017年5月。

npm install freeng --save

从freeng模块中导入(使用这种方式导入，会将所有的模块都导入)：

import {AccordionModule} from 'freeng/freeng'; //accordion @NgModule({ declarations: [AppComponent, ...], imports: [AccordionModule, ...], bootstrap: [AppComponent] }) export class AppModule {}
