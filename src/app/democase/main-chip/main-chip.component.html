碎片可以用来表示小块信息，它们最常用于联系人或标记。

API 选项

free-chip-group

碎片组

属性 类型 描述 chips Array 碎片数组 placeholder String 设置input输入框的提示文本

Import

import {ChipModule} from 'freeng/freeng';

Default

<free-chip-group [chips]="animals"></free-chip-group>

Javascript

this.animals = [ {'value': 'dog'}, {'value': 'cat'}, {'value': 'wolf'}, {'value': 'fish'} ];

Delete

<free-chip-group [chips]="fruit"></free-chip-group>

Javascript

this.fruit = [ {'value': 'apple'}, {'value': 'banana', 'delete': true}, {'value': 'orange', 'delete': true} ];

Input

<free-chip-group [chips]="city" placeholder="Add a chip"></free-chip-group>

Javascript

this.city = [ {'value': 'beijing'}, {'value': 'guangzhou', 'delete': true}, {'value': 'shanghai', 'delete': true}, {'value': 'hunan', 'delete': true} ];
