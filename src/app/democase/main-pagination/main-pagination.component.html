分页器

API 选项

free-pagination

采用分页的形式分隔长列表

属性 类型 描述 maxPage Number 最多显示页数，可选。默认5. row Number 每页显示的数据数，可选，默认为10 activeIndex Number 当前页码，可选。

事件

事件 描述 onPageChange 每次点击触发，返回参数类似： {{'{activeIndex: 1}'}}

Import

import {PaginationModule} from 'freeng/freeng';
