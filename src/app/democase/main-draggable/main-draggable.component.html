可拖放元素的控件。

API 选项

free-draggable

拖放控件

属性 类型 描述 dragItems Array 拖放列表数组 dropTarget - 拖放目标元素 draggable Boolean 列表是否可拖放，默认为false dragEffect string 拖放效果，默认为copy。可选值(copy/move)

事件

属性 类型 描述 onDragStart Function 拖放开始时触发的回调函数 onDragEnd Function 拖放结束时触发的回调函数

Import

import {DraggableModule} from 'freeng/freeng';
