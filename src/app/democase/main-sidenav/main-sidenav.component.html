侧边栏以滑动方式展现菜单或导航等内容。

API 选项

free-sidenav

侧边栏。

属性 类型 描述 visible Boolean 是否展开，必选。默认为false direction String 设置侧边栏的位置。默认为left。可选(left/right) overlay Boolean 是否显示半透明背景(可点击关闭侧边栏)，可选。默认为false

事件

事件 类型 描述 onChange Function 当侧边栏关闭或打开时触发的回调函数，可选。返回值: {open: true | false}

Import

import {SidenavModule} from 'freeng/freeng';
