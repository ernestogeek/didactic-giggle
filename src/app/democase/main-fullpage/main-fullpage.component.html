全屏切换一般用于宣传页面。

API 选项

free-fullpage

全屏滚动容器

属性 类型 描述 pagination Boolean 是否显示分页器，默认为false，可选 paginationClickable Boolean 分页器是否可点击，默认为false，可选

free-fullpage-slide

全屏滚动容器的子组件，需要展示的内容都填进子组件中。

Import

import {FullpageModule} from 'freeng/freeng';
