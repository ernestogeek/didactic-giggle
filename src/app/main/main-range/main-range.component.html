范围控件允许用户通过沿滑杆或轨道移动滑块旋钮从一系列值中进行选择。

API 选项

free-range

单个范围控件

属性 类型 描述 value Number 设置当前值。 min Number 设置最小值，默认为0 max Number 设置最大值，默认为100 color String 设置主题色

Default

<free-range></free-range>

Min and Max

<free-range [min]="10"></free-range> <free-range [min]="-5" [max]="80"></free-range>

Theme

<free-range [value]="20" color="red"></free-range> <free-range [value]="20" color="lightblue"></free-range> <free-range [value]="20" color="green"></free-range> <free-range [value]="20" color="yellow"></free-range>
