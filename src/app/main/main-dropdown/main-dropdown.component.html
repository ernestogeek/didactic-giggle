下拉菜单可以帮助你节省空间，同时提供更多选择

API 选项

free-dropdown

单个下拉菜单

属性 类型 描述 menus Array 菜单子项数组 direction String 设置下拉菜单展开的位置(bottom-left, bottom-right)， 可选，默认为bottom-left。 hover Boolean 设置鼠标移动上去时展开，可选，默认为false color String 设置主题样式，可选

Default

Dropdown
<free-dropdown [menus]="menus" color="green"> <f-header>Dropdown</f-header> </free-dropdown>

Direction

Dropdown
<free-dropdown [menus]="menus" direction="bottom-right" color="green"> <f-header>Dropdown</f-header> </free-dropdown>

Hover

Dropdown
<free-dropdown [menus]="menus" color="lightblue" [hover]="true"> <f-header>Dropdown</f-header> </free-dropdown>
