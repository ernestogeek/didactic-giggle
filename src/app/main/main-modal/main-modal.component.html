模态框包含专门针对特定任务的文本和UI控件。 它们向用户通知关键信息，要求用户作出决定或涉及多个任务

API 选项

free-modal

单个模态框。

属性 类型 描述 header String 模态框标题。 visible String 是否显示模态框，默认为false modalColor String 设置模态框的背景色。
