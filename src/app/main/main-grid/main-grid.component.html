FreeNG框架是使用基于flex显示属性的网格系统。

  • 响应式修饰符可以在xs、sm、md、lg四个视口宽度上指定不同的列大小，偏移量，对齐和分布。
  • 通过“行（row）”在水平方向创建一组“列（column）”

栅格参数

超小屏幕手机
(<768px) 小屏幕平板
(≥768px) 中屏幕桌面显示器
(≥992px) 大屏幕大桌面显示器
(≥1200px) 类前缀 .col-xs .col-sm .col-md .col-lg .container 最大宽度 None （自动） 750px 970px 1170px 列（column）数 12 可嵌套 是 偏移（Offsets） 是

简单语法

所有你需要记住的是行，列，内容。

<div class="row"> <div class="col-xs-12"> <div class="box">12</div> </div> </div>
