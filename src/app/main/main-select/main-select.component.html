下拉选择框可在多个选择之间进行选择。选择显示当前状态和向下箭头。

API 选项

free-select

下拉选择框。

属性 类型 描述 options Array 设置option，必选 pholder String 设置可描述输入字段预期值的提示信息，可选 selected String 设置可描述输入字段预期值的提示信息，可选 multiple Boolean 是否可选择多个选项，可选 filter Boolean 是否显示过滤输入框，可选 事件 类型 描述 onChange Function 当元素的值发生改变时，会发生 onChange 事件。
