API 选项

free-shrink

一个浮动按钮组件。

属性 类型 描述 menus Array 必选，浮动按钮数组 type String 可选，设置浮动按钮的类型(vertical、vertical-reverse、horizontal、horizontal-reverse、circle)，默认horizontal direction String 可选，圆形浮动按钮的位置(lt、lb、rt、rb)，默认lt
